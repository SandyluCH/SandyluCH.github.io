#Karma
  Karma运行在Node.js的环境下，而且可作为一个npm package;
  ##安装
  1、搭建node.js环境
  2、安装karma及其相应的插件
  <pre>
    <code>
	 # Install Karma:
     $ npm install karma --save-dev
     # Install plugins that your project needs:
     $ npm install karma-jasmine karma-chrome-launcher jasmine-core --save-dev
	</code>
  </pre>
